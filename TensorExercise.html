Multiple Choice Quiz

1. Which of the following show the correct initialization of tensor? Note: All the libraries already included

The correct answers are A, B and C.

How do you create a tensor of zeros with a shape of (2, 3) in PyTorch?

Answer: B. torch.zeros(2, 3)

3. Which method converts a NumPy array to a PyTorch tensor?

Answer: B. torch.from_numpy()

4. What function would you use to determine the shape of a tensor in PyTorch?

Answer: A. tensor.size()

5. How do you perform element-wise addition of two tensors, a and b?

Answer: C. a + b

6. What is the correct way to move a tensor to the GPU in PyTorch?

Answer: B. tensor.cuda()

7. How do you create a tensor with random values sampled from a normal distribution in PyTorch?

Answer: B. torch.randn(2, 2)

8. Which method would you use to change the shape of a tensor x from (2, 3) to (3, 2)?

Answer: B. x.reshape((3, 2))

9. What does tensor.view(-1) do?

Answer: C. Flattens the tensor to a one-dimensional tensor

10. Which of the following is NOT a valid way to create a PyTorch tensor?

Answer: D. torch.create_tensor([1, 2, 3])

11. Which is the default data type of tensor?

Answer: A. 32-bit floating-point.

11. zero_() method of tensor modifies the original tensor or create a new output tensor or return it?

Answer: A. Modifies it
